搭建Hexo的方法

  • 安装Nodejs和npm

  • 切换到国内的npm源

  • 用cnpm安装hexo

  • 建立空文件夹

  • 用hexo初始化博客

Hexo使用

  • 建立新的博客

  • 运行博客

  • 生成静态文件

  • 推送到站点

部署到github

  • New repository
    Repository name必须是： username.github.io

  • 在博客目录下安装git插件

  • 设置_config.yml文件（注意冒号后面必须跟空格）

  • 部署hexo

中间会会提示输入github用户名和密码。

  • 至此便可以访问 username.github.io

更换hexo主题

在 https://hexo.io/themes/ 中可以找到很多主题，找到喜欢的主题在对应的github界面查看安装方法。
以主题yilia作为范例(https://github.com/litten/hexo-theme-yilia)

  • 将主题克隆到themes目录下

  • 修改博客目录下的_config.yml

  • hexo clean

  • hexo s
